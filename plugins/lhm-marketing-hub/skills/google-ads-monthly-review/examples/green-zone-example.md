## Your 5 Priority Actions (Low Urgency)

### 1. DOCUMENT SUCCESS: Record What's Working
**Priority**: This week

**Action**: Create a template/document noting:
- Current campaign structure
- Budget allocation percentages
- Negative keyword list from last month
- Top performing keywords and ads
- Any seasonal factors

**Reasoning**: When things work well, document it! This helps:
- Maintain consistency if you're away
- Train team members
- Apply learnings to similar clients
- Remember what changed if performance shifts


### 2. TEST NEW AD COPY: Low-Risk Experiments
**Priority**: This month

**Action**: Create ad variations for testing:
- Test 2-3 new headlines in Generic - Chiropractor
- Focus on different angles (convenience, expertise, conditions)
- Ensure all copy is AHPRA compliant
- Run A/B test at 50/50 split

**Reasoning**: Stable accounts are perfect for testing. If new ads
perform worse, the account absorbs the impact. If they perform better,
you've improved an already strong performer.


### 3. MONITOR COMPETITORS: Stay Alert
**Priority**: Weekly check

**Action**: Review competitive landscape:
- Note any new competitors in auction insights
- Check for new messaging in competitor ads
- Monitor impression share for changes
- Watch for CPCs creeping up

**Reasoning**: Green zone today doesn't mean green zone forever.
Competitors can appear or improve. Early detection prevents surprises.


### 4. PLAN NEXT MONTH: Anticipate Changes
**Priority**: End of this month

**Action**: Consider upcoming factors:
- Any seasonal changes? (Winter = more back pain?)
- Client promotions or offers?
- Budget changes for June?
- New services to advertise?

**Reasoning**: Proactive planning in Green zone beats reactive
scrambling in Red zone. Use the calm to prepare.


### 5. INCREMENTAL KEYWORD OPTIMISATION: Fine-Tuning
**Priority**: Bi-weekly

**Action**: Light keyword maintenance:
- Add negatives for any new irrelevant search terms
- Consider phrase match for top exact match performers
- Remove any severely underperforming keywords (if any)
- Check Quality Scores for opportunities

**Reasoning**: Even well-performing accounts have small improvements
available. Regular small optimisations compound over time.

## Key Takeaways from This Example

### What Made This Green Zone

1. **Budget on pace** - 119% is within acceptable variance
2. **All campaigns under target CPA** - Best is 56% of target
3. **Client satisfaction** - Happy with booking volume
4. **Recent improvements working** - Last month's negatives effective

### Why These Actions Were Recommended

1. **Documentation** - Preserve what's working for consistency
2. **Ad testing** - Use stability for low-risk experiments
3. **Competitor monitoring** - Early warning system
4. **Future planning** - Proactive beats reactive
5. **Light optimisation** - Compound small improvements

### Green Zone Best Practices

- **Don't over-optimise** - If it's working, small touches only
- **Document success** - Know why it's working
- **Use for testing** - Stable accounts absorb test risk
- **Monitor proactively** - Green can become Orange quickly
- **Free up time** - Focus energy on struggling accounts
- **Plan ahead** - Use calm periods for future preparation

### Time Management in Green Zone

The strategy session correctly identified that Green zone accounts should receive less time than Red/Orange accounts. This allows:

- More time for accounts that need help
- Sustainable workload across 14+ clients
- Strategic improvements rather than firefighting

### Capturing Future Intel

The user mentioned a June service launch - capturing this in the session ensures:

- It's documented and won't be forgotten
- Next strategy session will include it
- Planning can begin proactively
