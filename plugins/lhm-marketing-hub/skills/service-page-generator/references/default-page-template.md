# Default Service/Condition Page Template

This template defines the section structure for a service or condition page. Follow this order exactly. Each section maps to a visual block on the website.

---

## Section Order

### 1. META Block

| Field | Value |
|-------|-------|
| META Title | [Primary Keyword] Support in [Location] | [Business Name + Services] |
| META Description | [150-160 chars with primary keyword, value prop, CTA] |
| Target Keywords | [comma-separated] |
| Slug | /conditions/[slug] or /services/[slug] |
| Existing Page | [URL or N/A] |

### 2. Header / Hero

```
[h2] [Condition/Service Name] Support at [Business Name]
[h1] [Condition/Service Name]

[p] 2-4 paragraphs. Acknowledge the reader's situation. State what the business
does. Set the tone. Primary keyword in first 100 words.

[Button] Book Now
[Button] Call [phone number]
```

### 3. What Is [Condition] / Common Symptoms

```
[h2] What Is [Condition Name] and What Are the Common Symptoms?

[p] Plain-language definition. Prevalence stats with citation.
Reassurance where appropriate.

[h3] Common Symptoms
- Bullet list of symptoms (5-7 items)

[h3] [Variant Symptoms, e.g. Nerve-Related Symptoms] (if applicable)
- Bullet list
- Red flag warning if clinically relevant

[Button] Book Now
```

### 4. Causes / Why This Happens

```
[h2] Common Causes of [Condition Name]

**Cause 1 name** — 2-4 sentence explanation
**Cause 2 name** — 2-4 sentence explanation
**Cause 3 name** — 2-4 sentence explanation
[Continue as needed, usually 4-6 causes]

[p] Closing paragraph connecting causes to who is most affected
```

### 5. Types / Variations

```
[h2] Types of [Condition Name]

**Type 1** — 2-3 sentence description
**Type 2** — 2-3 sentence description
[Continue as needed]

[Button] Book Now
```

### 6. FAQ Section

```
[h2] Frequently Asked Questions About [Condition Name]

**Q1?**
Answer (40-80 words)

**Q2?**
Answer (40-80 words)

[Minimum 5 questions, maximum 8]
```

### 7. Related Articles Widget

```
[h2] [Condition Name] Articles

[Blog post preview] | [Blog post preview] | [Blog post preview]
```

Placeholder section. Populated once supporting blog content is published.

### 8. How [Business] Can Help

```
[h2] How [Business Name] Treats [Condition Name]

[p] Opening paragraph on treatment philosophy. Citation to clinical guidelines.

[h3] Assessment
[p] What the assessment involves. Specific tests, tools, equipment.

[h3] Hands-On Treatment (if applicable)
[p] Manual therapy, techniques used. Position it within active rehab.

[h3] Exercise / Active Therapy
[p] Individualised exercise approach. What gets targeted and why.

[h3] [Specific Service, e.g. Clinical Pilates] (if applicable)
[p] How this service works at the business specifically.

[h3] Education and Self-Management
[p] What the patient/client learns. Practical strategies.

[h3] Return to Activity (if applicable)
[p] Programs for getting back to sport, work, daily life.

[Button] Book Now
[Button] Call [phone number]
```

### 9. Risk Factors (optional)

```
[h2] Who Is Most at Risk of [Condition Name]?

**Modifiable risk factors:**
- Bullet list

**Non-modifiable risk factors:**
- Bullet list

[p] Closing reassurance about modifiable factors
```

### 10. Lifestyle / Self-Management (optional)

```
[h2] What You Can Do at Home for [Condition Name]

**Tip 1 heading.** Explanation.
**Tip 2 heading.** Explanation.
[4-6 tips]
```

### 11. When to Seek Help

```
[h2] When to See a [Practitioner Type] for [Condition Name]

- Trigger 1
- Trigger 2
- Trigger 3

[p] No referral needed (if applicable in the practitioner's country)

[h3] When It May Be Serious (if clinically relevant)
- Red flag 1
- Red flag 2
[p] Brief note on urgency
```

### 12. Service Location CTA

```
[h2] Get Support for [Condition Name]
[h3] [Practitioner Type] [Location]

[p] If [condition] is affecting [daily life impact], early support helps.
[Business Name] is located in [Location] and services [surrounding areas].

[Button] Book Now
[Button] Call [phone number]
```

### 13. Footer CTA

```
Not Sure If This Is the Right Page?

You don't need a confirmed diagnosis to book an appointment.
If you're experiencing [general symptoms], our team can help assess
your symptoms and guide you in the right direction.

[Button] Book Now
[Button] [All Services/Conditions Page]
```

### 14. Schema Markup

Include at bottom of file:
- FAQPage JSON-LD (from FAQ section)
- LocalBusiness or MedicalBusiness JSON-LD (from client profile)
