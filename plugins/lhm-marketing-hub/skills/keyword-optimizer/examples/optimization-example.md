## Key Takeaways from This Example

### What Made This Ad Group Problematic

1. **60% wasted spend** - Majority of budget going to non-converters
2. **DIY intent keywords** - "exercises" attracted wrong audience
3. **Broad match too broad** - "lower back pain" matching everything
4. **No negatives** - Common waste patterns not blocked

### Why These Actions Were Recommended

1. **Pause high-waste keywords** - Immediate savings, no conversion loss
2. **42 negatives** - Block identified waste patterns
3. **Match type restriction** - Control what triggers ads
4. **Protect top performers** - Don't damage what's working

### Keyword Optimization Best Practices

- Start with search terms report - it shows real waste
- Group negatives by pattern (DIY, jobs, research, etc.)
- Add negatives before changing match types
- Protect your winners while cutting losers
- Monitor for 7 days after changes
- Document changes for client reporting

### Pattern Recognition for Negatives

The example identified 6 waste patterns:
1. DIY/Exercise seekers
2. Research/Information seekers
3. Job/Career seekers
4. Irrelevant conditions (pregnancy, emergency)
5. Product/Price shoppers
6. Wrong service comparisons

These patterns apply to most healthcare accounts.
