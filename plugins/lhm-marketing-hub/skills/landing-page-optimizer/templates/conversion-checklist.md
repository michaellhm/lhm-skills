# Landing Page Conversion Checklist

Comprehensive checklist for auditing healthcare landing pages against conversion best practices.

---

## Category 1: Hero Section (Above Fold)

### Headline
- [ ] Addresses visitor's problem/need
- [ ] Includes location keyword (if local service)
- [ ] Clear and easy to understand
- [ ] Not generic ("Welcome to..." is weak)
- [ ] AHPRA compliant (no guarantees/superlatives)

### Value Proposition
- [ ] Clear within 5 seconds of landing
- [ ] Answers "What's in it for me?"
- [ ] Differentiates from competitors
- [ ] Relevant to search intent

### Primary CTA
- [ ] Visible without scrolling
- [ ] Button stands out (contrasting colour)
- [ ] Action-oriented copy (not just "Submit")
- [ ] Single clear action (not multiple competing CTAs)

### Trust Signals
- [ ] AHPRA registration visible
- [ ] Practitioner credentials shown
- [ ] Professional association logos
- [ ] Google rating or reviews snippet

### Visual Elements
- [ ] Relevant hero image
- [ ] Professional quality photos
- [ ] Images support the message
- [ ] Not stock photos that feel generic

### Load Speed
- [ ] Page loads in under 3 seconds
- [ ] Images optimised
- [ ] No slow-loading elements blocking content

**Hero Section Score: ___ / 20**

## Category 4: Call-to-Action

### Primary CTA
- [ ] One clear primary action
- [ ] Button is prominent and obvious
- [ ] CTA copy is specific (not generic)
- [ ] AHPRA compliant (consultation-focused)

### CTA Copy Quality
- [ ] Action verb used ("Book", "Call", "Get")
- [ ] Low-friction language
- [ ] Benefit hinted ("Book Your Assessment")
- [ ] Urgency without pressure (AHPRA compliant)

### Multiple CTAs
- [ ] CTA above fold
- [ ] CTA after key sections
- [ ] CTA at bottom of page
- [ ] CTAs are consistent (same message)

### Contact Options
- [ ] Phone number prominently displayed
- [ ] Phone is click-to-call on mobile
- [ ] Email or form option available
- [ ] Hours of operation visible

**Call-to-Action Score: ___ / 15**

## Category 5: Forms

- [ ] Only essential fields requested

- [ ] Name, phone, message is often enough
- [ ] Email optional or clearly explained why needed

### Field Design
- [ ] Clear labels above or inside fields
- [ ] Required fields marked
- [ ] Error messages helpful
- [ ] Auto-fill works correctly

### Mobile Form Usability
- [ ] Fields are large enough to tap
- [ ] Appropriate keyboard shows (phone keyboard for phone)
- [ ] Form doesn't require scrolling horizontally
- [ ] Submit button is easily tappable

### Form Trust
- [ ] Privacy statement or link nearby
- [ ] What happens after submission is clear
- [ ] No surprising checkboxes
- [ ] Confirmation page/message works

## Quick Audit (5-Minute Version)

If you only have 5 minutes, check these critical items:

1. **Headline** - Does it address the visitor's problem?
2. **CTA Above Fold** - Can they book without scrolling?
3. **Phone Number** - Is it visible and clickable?
4. **Mobile Test** - Does it work on your phone?
5. **AHPRA** - Any obvious compliance issues?

These five items have the biggest impact on conversion rate.

## Common Healthcare Landing Page Issues

| Issue | Impact | Quick Fix |
|-------|--------|-----------|
| Generic headline | High | Lead with pain point |
| CTA below fold | High | Add CTA to hero |
| No phone number | High | Add prominent phone |
| Too many form fields | Medium | Remove non-essential |
| No social proof | Medium | Add testimonials |
| AHPRA violations | Critical | Remove immediately |
| Slow load | Medium | Optimise images |
| Poor mobile | High | Mobile-first redesign |
