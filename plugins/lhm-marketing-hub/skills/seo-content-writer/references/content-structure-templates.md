# Content Structure Templates

Markdown templates for common SEO content types.

## Blog Post (Informational)

**Target**: 1,200-1,800 words | **Goal**: Educate, build authority, rank for informational queries

```
# [Primary Keyword] - [Benefit or Hook]

[Hook - 1 sentence] → [Problem statement] → [Promise] → [Overview bullets]

## What Is [Topic]?
[Definition in 40-60 words — GEO-optimized] → [Expanded explanation] → [Why it matters]

## Why [Topic] Matters
[Benefit 1 with evidence] → [Benefit 2] → [Benefit 3]
> **Key Insight**: [Highlighted stat or point]

## [Secondary Keyword Section 1]
### [Sub-topic 1] → ### [Sub-topic 2]

## [Secondary Keyword Section 2]
[Table for comparisons/data]

## Common Mistakes to Avoid
### [Mistake 1] → ### [Mistake 2] → ### [Mistake 3]

## Frequently Asked Questions
### [PAA question]? → [40-60 word answer]

## Conclusion
[Recap] → [Final insight] → [CTA]
```

**Links**: 3-5 internal, 2-3 external | **Keywords**: Primary in H1, first 100 words, conclusion; secondary in H2s

## Comparison Article ("[A] vs [B]")

**Target**: 1,500-2,500 words | **Goal**: Help readers choose, rank for comparison queries

```
# [A] vs [B]: Which Is Better for [Use Case]?

[Hook] → [Context] → [Promise]
**Quick answer**: [Brief verdict]

## Quick Comparison Table
| Feature | [A] | [B] |

## What Is [A]? → Pros/Cons
## What Is [B]? → Pros/Cons

## Detailed Comparison
### [Category 1] → Winner: [A/B]
### [Category 2] → Winner: [A/B]

## Pricing Comparison
## Which Should You Choose?
### Choose [A] if: → ### Choose [B] if:

## FAQ → ## Final Verdict
```

## Listicle ("Top N [Items]")

**Target**: 1,500-2,500 words | **Goal**: Curated recommendations, "best [topic]" queries

```
# [Number] Best [Items] for [Purpose] ([Year])

[Hook] → [Selection methodology]

## Quick Summary Table
| Rank | Item | Best For | Price |

## 1. [Item] - Best for [Use Case]
**What it is** → **Why we love it** → **Key features** → **Pricing** → **Pros/Cons**

## 2. [Item] - Best for [Use Case]
[Same structure]

## Comparison Table
## How to Choose → ## FAQ → ## Our Top Pick
```

## How-To Guide

**Target**: 1,200-2,000 words | **Goal**: Teach a process, "how to [task]" queries

```
# How to [Goal]: [Number]-Step Guide

[Hook] → [Problem] → [Promise]

## What You'll Need
**Tools**: → **Time**: → **Skill level**:

## Step 1: [Action]
[Instruction] → [Why important] → **How to do it**: 1. 2. 3.
> **Pro Tip**: [tip]

## Step 2: [Action]
[Continue pattern]

## Troubleshooting
### Problem: [Issue] → **Solution**: [Fix]

## Advanced Tips → ## FAQ → ## Conclusion
```

## Pillar Page (Comprehensive Guide)

**Target**: 3,000-5,000+ words | **Goal**: Definitive resource, head terms, topic cluster hub

```
# [Topic]: The Complete Guide ([Year])

[Hook] → [Challenges] → [Promise]

## Table of Contents

## Chapter 1: Fundamentals
### What Is [Topic]? → ### Why It Matters → ### Key Terms

## Chapter 2-5: [Major Sub-Topics]
[Deep sections with H3s, data, examples, internal links to cluster content]

## FAQ (8-10 questions)
## Key Takeaways → ## What's Next? → ## Resources
```

**Links**: 8-15 internal (cluster), 5-10 external | **Update**: Quarterly

## Implementation Checklist

For any template:
- [ ] Primary keyword in H1 and first 100 words
- [ ] Secondary keywords in H2/H3 headings
- [ ] 3-5 internal links
- [ ] 2-3 external authoritative links
- [ ] GEO-optimized definition blocks
- [ ] FAQ section for featured snippets
- [ ] Meta description 150-160 chars
- [ ] Title tag 50-60 chars
