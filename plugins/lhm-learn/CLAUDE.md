# LHM Learn - Plugin-Wide Rules

This plugin captures session learnings and writes them to the correct locations. It operates across all LHM plugins.

## Core Principles

1. **Never fabricate learnings** — only capture what actually happened in the session
2. **Skill learnings go to LEARNED.md** — patterns, gotchas, and workflow improvements that are reusable across sessions and clients
3. **Client learnings go to client_profile.md** — specific facts about the client's business, accounts, preferences, and configuration
4. **Always confirm before writing** — present findings to the user and get approval before modifying any files
5. **Respect existing content** — when updating client_profile.md, enrich and append, never overwrite existing data unless correcting an error
